# Reference implementation of the medical deconfounder

This repository contains the code for 

+ Two-cause simulation (Section 3.2)

+ Multi-cause simulation (Section 3.3)


# System requirement
1. Python 2.7
2. Edward 1.3.5
3. tensorflow 1.5.0
4. R 3.6.0
5. rstan 2.19.2
6. rstanarm 2.19.2
